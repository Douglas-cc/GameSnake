## Objetivo
Este repositório tem objetivo de exercitar a lógica de programação de forma lúdica, através do desenvolvimento de joguinhos 2D e coisas afins

## Requisitos

Você precisará de Python 3 e pip para instalar o pygame

```bash
$ pip install pygame
```
## Como executar?

```bash
$ python3 snake.py 
```
